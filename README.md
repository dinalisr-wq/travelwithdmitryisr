# Travel консультант

Одностраничный сайт travel-консультанта: услуги, направления, отзывы и форма заявки.

## Запуск

Откройте `index.html` в браузере (двойной клик по файлу) или запустите локальный сервер:

```bash
npx serve .
```

## Структура

- `index.html` — разметка
- `styles.css` — стили
- `script.js` — меню, анимации, форма, слайдер отзывов

## Настройка

Замените контакты, цены и ссылки на соцсети в `index.html`. Фотографии подгружаются с Unsplash (нужен интернет).
